# E-Ticaret Satıcı Yönetim Paneli

Bu proje, satıcıların ürün, sipariş ve kargo verilerini yönetebildiği bir dashboard uygulamasıdır. Toplu işlemler için Excel içe/dışa aktarma desteklenir.

## Özellikler

Backend (ASP.NET Core Web API)
- Firebase Firestore entegrasyonu
- Ürün yönetimi (CRUD)
- Sipariş yönetimi
- Kargo takip yönetimi
- Excel ile ürün/kargo içe aktarma ve dışa aktarma
- Swagger/OpenAPI
- Firebase Auth + HttpOnly cookie ile oturum yönetimi

Frontend (React + TypeScript)
- Satıcı paneli ekranları (ürünler, siparişler, kargo, excel işlemleri)
- Firebase Auth ile giriş

## Gereksinimler
- .NET 9 SDK
- Node.js
- Firebase projesi + Firestore
